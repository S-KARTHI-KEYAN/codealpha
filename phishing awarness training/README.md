# 🛡️ Phishing Awareness Training Module

An interactive, console-based Python application designed to educate users
about phishing attacks, social engineering tactics, and cybersecurity best
practices — built as part of a cybersecurity internship project.

## 📋 Features

- **Educational Modules** covering:
  - What phishing is and its various types (email, spear, smishing, vishing, whaling, clone phishing)
  - How to recognize phishing emails and fake websites
  - Common social engineering tactics used by attackers
  - Best practices and tips to avoid falling victim
  - Real-world phishing attack case studies
- **Interactive Quiz** with randomized, scored multiple-choice questions and instant feedback/explanations
- **Simple menu-driven interface** — no external dependencies required

## 🚀 Getting Started

### Prerequisites
- Python 3.6 or higher (no external libraries needed)

### Installation
```bash
git clone https://github.com/<your-username>/phishing-awareness-training.git
cd phishing-awareness-training
```

### Usage
```bash
python3 phishing_training.py
```

Follow the on-screen menu to navigate through training modules or jump
straight to the quiz.

## 🎯 Learning Outcomes
After completing this training, users will be able to:
- Identify common red flags in phishing emails and fake websites
- Recognize manipulation tactics used in social engineering

- Understand the real-world impact of phishing through case studies
